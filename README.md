# StandUpTimer

Программа-напоминание, которая каждые 90 минут выводит окно с призывом встать и подвигаться.

## Предыстория

Идея возникла в процессе лечения хронического простатита. Часовое сидение за компьютером создаёт застой крови в нижнем отделе позвоночника, что сводит "на нет" эффект от медикаментов. Заметил, что могу работать несколько часов подряд, даже ни разу не вставая.

Решил сделать небольшую тулзу, которая будет напоминать о необходимости двигаться. Заодно вспомнил C#, с которым не работал около 5 лет. С помощью четырёх запросов к ИИ получился рабочий инструмент.

## Принцип работы

- Приложение висит в трее
- Каждые 90 минут появляется белое окно во весь экран с текстом "ВСТАНЬ И ПРОЙДИСЬ"
- Окно не имеет звукового сопровождения
- После закрытия окна таймер сбрасывается и отсчитывает новые 90 минут
- При выключении компьютера программа завершается без следа

## Структура проекта

### Core

| Файл                     | Назначение                         |
| ------------------------ | ---------------------------------- |
| `AppState.cs`            | Модель состояния приложения        |
| `NotificationService.cs` | Отвечает за показ окна-напоминания |
| `TimerManager.cs`        | Управляет таймером и событиями     |

### UI

| Файл          | Назначение                     |
| ------------- | ------------------------------ |
| `TrayIcon.cs` | Иконка в системном трее с меню |

### Корень

| Файл               | Назначение               |
| ------------------ | ------------------------ |
| `AppController.cs` | Связывает все компоненты |
| `Program.cs`       | Точка входа              |
